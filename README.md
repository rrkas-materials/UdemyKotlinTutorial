# Kotlin Android App  

## Kotlin Basics

all basic topics are covered for starting journey with android using kotlin.

## App1: Find My Age

A simple application to input DOB and show age on button trigger.  

Key components used:  
 1. EditText
 2. TextView
 3. Button

## App2: Tik Tak Toy

A simple game application based on Tik Tak Toy.  

Key components used:  
 1. TableView
 2. TableRow
 3. Button
 4. TextView  

Features:  
 - 2 players
 - reset button
 - colors for players

## App3: Calculator  

A simple calculator with:  
 - plus/minus
 - percentage
 - AC  

Key comonents used:  
 1. TableView
 2. TableRow
 3. Button
 4. TextView
