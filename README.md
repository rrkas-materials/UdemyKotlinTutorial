# Kotlin Android App  

## 00 kotlin basics

all basic topics are covered for starting journey with android using kotlin.

## app1: Find My Age

A simple application to input DOB and show age on button trigger.  

Key components used:  
 1. EditText
 2. TextView
 3. Button

## app2: Tik Tak Toy

A simple game application based on Tik Tak Toy.  

Key components used:  
 1. TableView
 2. TableRow
 3. Button  

Features:  
 - 2 players
 - reset button
 - colors for players
